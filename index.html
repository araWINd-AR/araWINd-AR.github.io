Fall Detection & Protection by Airbag System

Arduino Embedded C Sensors Accelerometer IEEE Publication

This project is a wearable safety system made to detect sudden falls and reduce injury using an airbag system. It uses sensors to monitor body movement, sends the data to a microcontroller, and activates the airbag when fall-like movement is detected.

  • Built this project mainly to help elderly people by reducing the injury impact during sudden falls
  • Used motion sensors / accelerometer to monitor body movement and detect sudden changes
  • Connected the sensor data to a microcontroller like Arduino for checking fall conditions
  • Used threshold-based logic to decide whether the movement looks like a real fall
  • Added an airbag protection idea where the airbag inflates quickly after a fall is detected
  • Used a solenoid valve and compressed air concept to trigger the airbag deployment mechanism
  • Worked on both hardware and software parts, including sensor reading, embedded logic, and triggering control
  • Built this as a learning project to understand sensors, embedded systems, hardware integration, and safety technology
  • Published the related work through IEEE ICOECA 2024 as part of the project outcome
